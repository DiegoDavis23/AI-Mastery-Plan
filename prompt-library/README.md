# Atomic Digital — Prompt Library

> 10 reusable AI prompts for SEO, content marketing, ad copy, and email sequences.
> Each prompt follows the 5-element framework: Role, Context, Task, Format, Constraints.

## Prompts

| # | Name | Use Case |
|---|------|----------|
| 1 | SEO Meta Description | Write meta descriptions for any page |
| 2 | Blog Post Outline | Generate SEO-optimized blog structure |
| 3 | Blog Post Writer | Write a full blog post from an outline |
| 4 | Google Ad Copy | Create search ad variations |
| 5 | Facebook/Instagram Ad Copy | Social media ad copy |
| 6 | Email Welcome Sequence | 3-email onboarding sequence |
| 7 | Cold Outreach Email | Sales email for prospecting |
| 8 | Social Media Repurposer | Turn blog posts into social content |
| 9 | SEO Keyword Clustering | Group keywords by intent |
| 10 | Client Report Summary | Turn analytics data into insights |

## How to Use

1. Copy the prompt from the relevant `.md` file
2. Replace the `{{variables}}` with your client's info
3. Paste into Claude (or any AI) and run
4. Adjust temperature based on the recommended setting in each prompt
