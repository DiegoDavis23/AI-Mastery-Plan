## Week 1: Foundations & Claude Code Power Usage

### Day 1-2: Claude Code Deep Dive
- Install Claude Code, configure your environment
- Practice: Use Claude Code to scaffold a Next.js project from scratch
- Learn every slash command (/help, /compact, /clear, /init)
- Project: Build a simple landing page entirely through Claude Code prompts

### Day 3-4: Prompt Engineering Mastery
- Study system prompts, temperature, structured outputs
- Project: Create a prompt library for your agency - write 10 reusable prompts for SEO meta descriptions, blog outlines, ad copy, email sequences
- Store them in a git repo managed through Claude Code

### Day 5-6: Claude API Fundamentals
- Get API access, learn the Messages API, tool use, streaming
- Project: Build a CLI tool that takes a URL, scrapes the page, and generates an SEO audit using Claude API

### Day 7: Review & Document
- Write up what you learned in a personal knowledge base
- Refine your prompt library based on output quality

---

## Week 2: Building AI-Powered Marketing Tools

### Day 8-9: Content Generation Pipeline
- Project: Build a blog post generator that takes a keyword, researches competitors (via web scraping), and produces a full SEO-optimized article
- Use Claude Code to write the entire pipeline

### Day 10-11: SEO Automation
- Learn programmatic SEO concepts
- Project: Build a tool that takes a CSV of keywords, generates unique landing pages for each, with meta tags, schema markup, and internal linking suggestions

### Day 12-13: Social Media Content Engine
- Project: Build a system that takes one long-form blog post and outputs 10 Twitter posts, 5 LinkedIn posts, 3 Instagram captions, and 1 email newsletter draft
- Use Claude's structured output (JSON mode) to format everything

### Day 14: Integration Day
- Connect your tools together into a single workflow
- Add error handling, logging, and output to markdown/HTML files

---

## Week 3: Advanced AI Patterns & Agency Infrastructure

### Day 15-16: Multi-Step Agents
- Study tool use and agentic patterns in the Claude API
- Project: Build an AI agent that can research a topic, outline content, write it, self-critique, and revise - all autonomously in a loop

### Day 17-18: RAG (Retrieval-Augmented Generation)
- Learn embeddings, vector search basics
- Project: Build a knowledge base for your agency - ingest client briefs, brand guidelines, past content - then query it to generate on-brand content

### Day 19-20: Client Reporting Dashboard
- Project: Build a tool that takes Google Search Console / Analytics data (CSV exports to start), feeds it to Claude, and generates a written performance report with recommendations
- Output as styled HTML or PDF

### Day 21: MCP Servers
- Learn Model Context Protocol - how to give Claude Code access to custom tools
- Project: Create a simple MCP server that exposes your SEO tools to Claude Code directly

---

## Week 4: Productize & Ship

### Day 22-23: Build Your Agency's Internal Platform
- Project: Combine weeks 1-3 into a single web app (Next.js or Python/FastAPI)
- Features: content generation, SEO audit, content repurposing, client reporting
- Use Claude Code to build the full stack

### Day 24-25: Workflow Automation
- Connect your tools to real services: Google Sheets API, WordPress API, email sending
- Project: Build an end-to-end flow - keyword goes in, published blog post comes out

### Day 26-27: Fine-Tune Your Process
- Document your agency's AI workflows as SOPs
- Project: Create a Claude Code CLAUDE.md project file that encodes your agency's style guide, tone, SEO rules so every Claude session follows your standards

### Day 28: Portfolio & Launch
- Package 3 case studies showing AI-generated results
- Set up your agency site with the tools you built
- Write a LinkedIn post breaking down your process
